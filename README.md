# 🐍 贪吃蛇 - Snake Game

一个功能丰富的网页版贪吃蛇游戏，纯 HTML/CSS/JS 实现，零依赖，打开即玩。

## ✨ 功能特色

### 核心玩法
- 经典贪吃蛇规则：方向键 / WASD 控制移动，吃食物变长
- 撞墙或撞到自己 → 游戏结束
- 空格 / 回车快速开始或重新开始

### 音效系统
- **背景音乐**：8-bit chiptune 风格循环旋律（130 BPM）
- **6 种音效**：
  - 🎵 开局上行琶音
  - 🎵 转向轻微点击声
  - 🎵 吃食物双音阶叮咚
  - 🎵 里程碑庆祝四连音（每 50 分）
  - 🎵 死亡四阶下行悲壮音
  - 🎵 通关胜利七音阶号角
- 🔇 右上角静音开关按钮

### 特殊食物
- ⭐ 随机出现的金色六角星形食物
- 吃掉后 **+30 分** + 接下来 8 步**速度减慢**
- 金色粒子爆炸特效
- 出现时间过长会闪烁提醒

### 游戏机制
| 功能 | 说明 |
|------|------|
| 难度等级 | 每 100 分升 1 级，header 显示 |
| 速度递进 | 每 50 分加速一次 |
| 最高分 | localStorage 持久化存储 |
| 暂停 | 按 `P` 或 `Esc` 暂停/恢复 |
| 通关 | 占满所有格子即为胜利 |

### 视觉特效
- 蛇身绿色渐变，蛇头发光 + 方向眼睛
- 食物径向渐变 + 脉冲动画
- 吃食物粒子爆炸效果
- 暗色背景 + 网格线
- 待机画面游动装饰蛇

### 移动端适配
- 📱 触屏滑动手势控制方向
- 📱 手机屏幕自动显示虚拟方向键
- 📱 响应式布局，自适应窗口大小

## 🚀 快速开始

1. 克隆项目或直接下载 `index.html`
2. 用浏览器打开 `index.html`
3. 点击「开始游戏」或按空格键开始

```bash
# 克隆仓库
git clone https://github.com/tel3/SnakeGame.git
cd SnakeGame

# 直接用浏览器打开
open index.html  # macOS
start index.html # Windows
```

或通过 GitHub Pages 在线游玩：
👉 **https://tel3.github.io/SnakeGame**

## 🎮 操作指南

| 按键 | 功能 |
|------|------|
| `↑ ↓ ← →` / `W A S D` | 控制蛇的移动方向 |
| `空格` / `回车` | 开始 / 重新开始 |
| `P` / `Esc` | 暂停 / 恢复 |

移动端直接**滑动屏幕**或点击**虚拟方向键**。
